# Fuloh's Quality of Life (QoL) Hub

**Fuloh's QoL** is a modular, all-in-one World of Warcraft addon designed to streamline your gameplay with essential quality-of-life improvements. Instead of managing multiple scattered addons, Fuloh's QoL consolidates them into a single, lightweight hub with a unified settings panel.

## Key Features

### 🛡️ Joined Group Reminder
Stop asking "which key is this?" after joining a group!
*   **Instant Context:** Automatically displays a banner with the dungeon name and group title when you join via LFG.
*   **Live Updates:** Refreshes automatically if the group leader changes the dungeon or restarts the listing.
*   **One-Click Teleports:** Includes a quick teleport button for the specific dungeon (if you have the Hero's Path spell).
*   **Smart Hiding:** Auto-hides when the key starts or when you leave the group.
*   **Customizable:** Draggable banner with position saving. Now 50% larger for better visibility!

### 👋 Hello World
Automate your social interactions without sounding like a bot.
*   **Auto-Greeting:** Automatically greets your party when you join.
*   **Human-like Behavior:** Adds a natural, randomized delay (4-6 seconds) before sending the message.
*   **Smart Channels:** Intelligently switches between Party and Instance chat.
*   **Customizable:** Define your own list of random greetings.

### 🏁 Auto GG (GGGuys)
Celebrate your timed keys with the group effortlessly.
*   **Auto-Congratulate:** Automatically says "GG" in party chat when a Mythic+ dungeon is completed in time.
*   **Human-like Reaction:** Includes a natural, randomized delay (4-6 seconds) after the key completion before sending the message.
*   **Customizable:** Define your own list of random congratulations messages to cycle through.

## Why Use Fuloh's QoL?
*   **Modular Design:** Enable only the features you need via the centralized settings panel. **All features are disabled by default** so you have full control.
*   **Localized:** Fully translated into French (frFR) including dungeon names and UI text.
*   **Lightweight:** Optimized for performance with minimal memory usage.
*   **User Friendly:** Simple commands (`/fuloh`) and an intuitive interface.

## Getting Started
Access the main hub with `/fuloh` or via the Interface Options menu to enable features.
